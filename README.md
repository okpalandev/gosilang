# GosiLang - Gossip Language.
"Don’t ask what your country can do for you, but what you can do for your country."
- Nnamdi Michael Okpala

GosiLang is a functional, Turing-complete programming language designed for native Igbo speakers, aimed at
empowering the Igbo community.

## Features
-    Functional Programming: GosiLang leverages functional programming concepts and the lambda calculus framework.
-   Simplicity and Efficiency: Introduces new tools and concepts to simplify programming, making it more accessible and efficient.
